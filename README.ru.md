# >>>[РУССКИЙ](https://github.com/windelicato/painterofthezone/blob/main/README.ru.md)&lt;&lt;&lt;

Высоко настраиваемый пересмотр улучшения боевых действий для S.T.A.L.K.E.R. ГАММА. с обширной конфигурацией MCM

# ФУНКЦИИ

### Тектуры Wallmark высокого разрешения с параллаксом и поддержкой материала

### Обширные варианты MCM, чтобы включить, отключить или настраивать все возможное

### Улучшения для стен двигателя

-   Когда поблизости нет стен.
-   16+ дополнительных вариаций Wallmark
-   Кровавый бассейн под целью смерти

### Усовершенствования для включения в частицы двигателя FX

-   Повторно
-   Реактивная система частиц с HIT FX, которая отображает обратную связь на основе:
    -   Броня Target (EXO, средняя доспеха, стеклянные шлемы, тканевая броня)
    -   Способность пирсинга игрока, рассчитанная по пуле, оружию и нацеливанию брони
    -   Часть тела ударила выстрелом (выстрел в голову, выстрел в боди)
    -   Потенциал убийства с одним ударом (массивный FX на одном ударе головы)

### Расчленение

-   Переработал мод @XCVB, чтобы не появляться сетки и иметь более низкий след
-   Возможность использовать стенку для тела вместо удаления конечностей, чтобы указать массивные попадания
-   Возможность разместить брызги крови и текстуры раны на стенах вокруг расчленения

### Время пули

-   Замедляет время в течение 2 с после приземления мощного выстрела в голову, чтобы выделить визуальный FX
-   Время может быть расширено, приземлившись на несколько последовательных выстрелов в голову
-   Вознаграждение игрока за точность, позволяя легко нанести последующие снимки

### камера убийства / Kill cam

-   Замедление движения убийственная кулака, которая перемещает камеру в местоположение цели на убийстве
-   Многие варианты для запуска, чтобы сделать его полезным:
    -   Только когда текущее оружие наносит гораздо больше урона, чем необходимо для убийства
    -   Только когда цель очень далеко
    -   Только когда цель - последний враг в области
    -   Только когда время пули продлевается определенным количеством раз

### Особенности безопасности

-   Избегайте какого -либо actor_on_update, если у пользователя не включено сглаживание времен
-   Критические или спамальные обратные вызовы задерживаются (т.е. дробовики ударили о обратных вызовах)
-   Уничтожение любых временных объектов хранения при сохранении / загрузке
-   Нет нереста игровых объектов или сетки, чтобы отслеживать или удалять в случае удаления

# Доступны настройки MCM

### Стены

-   Включить / отключить усовершенствования на стенах
-   Включить / отключить бассейны на смерти
-   Возможность сохранить размер стенки двигателя
-   Вариант сделать все стены одинакового размера
-   Включить / отключить дополнительные стены вокруг точки расчленения
-   Возможность установить расстояние от расчленения на стены
-   Опция установить количество дополнительных стен
    -   Первые 3 стены всегда будут включать в себя всплеск крови, скоростную текстуру раны и капли крови
-   Варианты выбора из 3 категорий стиля Wallmark:
    -   Нормальный: Bullet Hole Bloodmarks с тропами
    -   Drip: капли, как те, от кровоточащего игрока
    -   Splash: большие брызги крови
-   Возможность проверить на наличие близлежащих стен перед размещением
-   Возможность задержать размещение Wallmark, когда несколько попаданий приземляются в короткие
-   TTL для стен
-   Возможность установить гравитацию для пола. Выше приближает их к цели

### Частицы

-   Включить / отключить усовершенствования частиц
-   Включить / отключить систему реактивных частиц
-   Включить / отключить частицу при ударе
-   Включить / отключить заднее облако пыли и крови при ударе
-   Возможность выбрать направление крови облако
-   Возможность установить размер крови облако
-   Включить / отключить частицу на выстреле голова
-   Включить / отключить частицу на выстреле головного выстрела из -за порога расчленения
-   Варианты воспроизведения всех этих частиц за выстрел или только один
-   Включить / отключить ограничение одной из каждой частицы за раз
-   Возможность установить TTL для каждой частицы перед заменой

### Расчленение

-   Включить / отключить для сталкеров
-   Включить / отключить для мутантов
-   Включить / отключить только для голов
-   Включить / отключить стенку для тела вместо удаления конечности
-   Варианты получения порога для сталкеров и мутантов

### Время пули

-   Включить / отключить время пули
-   Возможность выбрать, сколько времени замедлится
-   Возможность выбрать, как долго продлится время пули
-   Возможность включить звук при активации / деактивации
-   Возможность включить эффекты наложения экрана во время пули
-   Включить / отключить временные изменения постепенно, а не мгновенно (отключите для Actor_on_update без))
    -   Опция установить, насколько постепенно меняется время
-   Параметры триггера:
    -   Использует порог расчленения с включенным расчленением
    -   Возможность установить плоскую шанс запустить на выстрел в голову убийство

### камера убийства / Killcam

-   Включить / отключить Killcam
-   Возможность выбрать, сколько времени замедлится
-   Вариант выбора, как долго будет продлиться Killcam
-   Возможность выбрать, насколько далеко камера от цели
-   Включить / отключить временные изменения постепенно, а не мгновенно (отключите для Actor_on_update без))
    -   Опция установить, насколько постепенно меняется время
-   Параметры триггера:
    -   Возможность запустить, когда Target находится на определенном расстоянии от игрока
    -   Возможность запустить только тогда, когда другие враги не сражаются с игроком на определенном расстоянии
    -   Возможность запустить только тогда, когда время пули было продлено определенным количеством раз
